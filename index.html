List of Some Selectors and Methods

console.log(string)

Outputs a log message to the Console panel.

console.table(["Car", "Bus", "Train"]);

The console.table() method writes a table in the console view.

console.error("Mistake at the line 10");

The console.error() method writes an error message to the console.

console.clear();

The console.clear() method clears the console. The console.clear() method will also write a message in the console: "Console was cleared".

Alert

Displays a dialog box that contains the string that’s passed to it by the parameter along with an OK button.

Prompt

Displays a dialog box that contains the string in the rst parameter, the default value in the second parameter, an OK button, and a Cancel button. If the user enters a value and clicks OK, that value is returned as a string. If the user clicks Cancel, null is returned.

Confirm?

Displays a modal dialog with an optional message and two buttons, OK and Cancel. It returns true if the user clicks “OK”, and false otherwise. It prevents the user from accessing other parts of the page until the box is closed.

parseInt

Converts the string it receives to an integer data type and returns that value. If it can’t convert the string to an integer, it returns NaN.

parseFloat

Converts the string it receives to a decimal data type and returns that value. If it can’t convert the string to a decimal value, it returns NaN.

querySelector

Returns the rst HTML element that matches the selector that’s passed to it. The parameter is a string that must contain a valid CSS selector.

querySelectorAll

Returns all the HTML elements that match the selector that’s passed to it. The parameter is a string that must contain a valid CSS selector.

toLowerCase()

Returns a new string with the letters converted to lowercase.

toUpperCase()

Returns a new string with the letters converted to uppercase.

toFixed

Rounds a number to n decimal places and converts it to a string.

Math.sqrt(number);

The sqrt() method returns the square root of a number

Math.min(5, 10);

The min() method returns the number with the lowest value

Math.max(5, 10);

The max() method returns the number with the highest value.

Math.pow(x, y)

The pow() method returns the value of x to the power of y (xy).

getFullYear

Returns the four-digit year from the date.

getDate

Returns the day of the month from the date.

getMonth

Returns the month number from the date. The months are numbered starting with zero (January is 0 and December is 11).

preventDefault()

Stops the default action for the event from occurring.

indexOf(search,position)

Searches for the rst occurrence of the search string starting at the position specfed or zero if position is omitted. If found, it returns the position of the rst character, counting from 0. If not found, it returns -1.

substr(start,length)

Returns the substring that starts at the specfed position (counting from zero) and contains the specfed number of characters.

substring(start,stop)

Returns the substring that starts at the specfed position (counting from zero) and stops at the specfed position (counting from zero).

str.replace()

The replace() method searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.

str.charAt(index)

The charAt() method returns the character at a specified index in a string. The index of the first character is 0, the second character is 1, and so on.

str.charCodeAt(index)

The charCodeAt() method returns the Unicode of the character at the specified index in a string. The index of the first character is 0, the second character 1, and so on.
